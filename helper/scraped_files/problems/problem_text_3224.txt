 CSES - Network Renovation      Syrjälä's network consists of nn computers and n−1n−1 connections between them. It is possible to send data between any two computers.  However, if any connection breaks down, it will no longer be possible to send data between some computers. Your task is to add the minimum number of new connections in such a way that you can still send data between any two computers even if any single connection breaks down.
